The static dataset lives in `src/data/pokemon.json`. The **server route** (`src/app/api/pokemon/route.ts`) handles all filtering, sorting, and pagination against this JSON — clients only receive the 20 records per page they need. The **client fetch layer** (`src/lib/api.ts`) builds query params and calls `/api/pokemon`. Swapping to a real database requires changes only in `route.ts`.
